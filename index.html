📖 5W2H 框架 · 彻底理解“类”

🔍 What（是什么）

类是创建对象的“蓝图”或“模板”。它定义了一类对象共有的属性（数据）和方法（行为）[reference:2]。

❓ Why（为什么）

封装数据与操作，提高代码复用性、可维护性和组织性。避免散落的变量和重复代码[reference:3]。

👤 Who（谁适用）

任何希望写出更清晰、可扩展代码的Python开发者，尤其是构建中大型项目时。

⏰ When（何时用）

需要创建多个具有相同属性和行为的对象时，例如管理多个学生、员工、游戏角色等。

📍 Where（哪里定义）

通常在模块（.py文件）的顶层定义，方便在程序的其他部分导入和使用。

🛠️ How（怎么用）

使用 class 关键字定义类，通过 __init__ 初始化属性，创建实例（对象）并调用其方法[reference:4]。

⚖️ How much（成本/收益）

初期有少量学习成本，但长期能显著提升代码质量、开发效率和团队协作能力。

🎨 图形化类比 · 类与对象的关系

🏠类 (Class)

就像一张建筑蓝图，定义了房子应该有哪些房间（属性）和功能（方法）。

➡️实例化

照着蓝图盖房子的过程。

🏡对象 (Object)

就是根据蓝图建出来的真实的房子。每个房子（对象）都是独立的，但都遵循同一个设计图（类）。

# 定义类 (蓝图) class House: def __init__(self, owner, area): self.owner = owner # 属性: 房主 self.area = area # 属性: 面积 def introduce(self): # 方法: 介绍自己 print(f"{self.owner}的房子，面积{self.area}平米") # 创建对象 (建房子) my_house = House("小明", 120) your_house = House("小红", 85) # 调用对象的方法 my_house.introduce() # 输出: 小明的房子，面积120平米 your_house.introduce() # 输出: 小红的房子，面积85平米

✍️ 练习题目 · 从入门到实战

❶ 【入门】创建一个 Book 类，具有 title 和 author 属性，并包含一个 info() 方法，打印“《书名》的作者是作者名”。创建两个实例并调用 info()[reference:5]。

✅ 参考代码
class Book:
    def __init__(self, title, author):
        self.title = title
        self.author = author
    def info(self):
        print(f"《{self.title}》的作者是{self.author}")

book1 = Book("三体", "刘慈欣")
book2 = Book("活着", "余华")
book1.info()  # 输出：《三体》的作者是刘慈欣
book2.info()  # 输出：《活着》的作者是余华

❷ 【进阶】创建一个 BankAccount 类，包含 account_holder 和 balance 属性。实现 deposit(amount) 和 withdraw(amount) 方法，并确保余额不能为负。最后打印账户信息[reference:6]。

✅ 参考代码
class BankAccount:
    def __init__(self, account_holder, balance=0):
        self.account_holder = account_holder
        self.balance = balance
    def deposit(self, amount):
        if amount > 0:
            self.balance += amount
            print(f"存款{amount}元，当前余额: {self.balance}")
    def withdraw(self, amount):
        if 0 < amount <= self.balance:
            self.balance -= amount
            print(f"取款{amount}元，剩余余额: {self.balance}")
        else:
            print("余额不足！")
    def show_info(self):
        print(f"{self.account_holder}的账户余额: {self.balance}元")

acc = BankAccount("张三", 500)
acc.deposit(200)
acc.withdraw(300)
acc.withdraw(500)
acc.show_info()

❸ 【挑战】创建一个 Student 类，并运用“继承”，创建一个 GraduateStudent 子类，为其增加 research_area 属性[reference:7]。

✅ 参考代码
class Student:
    def __init__(self, name, student_id):
        self.name = name
        self.student_id = student_id
    def introduce(self):
        print(f"我是学生{self.name}，学号{self.student_id}")

class GraduateStudent(Student):
    def __init__(self, name, student_id, research_area):
        super().__init__(name, student_id)
        self.research_area = research_area
    def introduce(self):
        print(f"我是研究生{self.name}，研究领域是{self.research_area}")

# 测试
g_stu = GraduateStudent("李华", "G2024001", "人工智能")
g_stu.introduce()
